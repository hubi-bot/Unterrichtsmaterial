Erinnerung – eine lineare Funktion sieht so aus: f(x) = m · x + b  ·  m = Steigung, b = y-Achsenabschnitt (Startwert).

Vom Text zur Mathematik

Jetzt der wichtige Schritt: Wie wird aus einer Textaufgabe eine Rechnung? Lies das genau – danach geht es in Level 2 los.

⭐ Diese zwei Wörter musst du kennen

▸ Funktionswert = y-Wert. Das ist das Ergebnis – der Wert auf der senkrechten Achse.
▸ „Wert" = x-Wert. Wenn eine Aufgabe sagt „berechne einen Wert", ist die gesuchte Stelle x gemeint – auf der waagerechten Achse.

Schritt 1 · Was ist x, was ist f(x)? immer zuerst!

▸ f(x) = das Ergebnis, das du wissen willst. Es hängt ab von etwas anderem.
▸ x = das, wovon das Ergebnis abhängt – das, was du einsetzt.
▸ Suchtrick: Finde im Text das Wort „pro" oder „je". Direkt danach steht die Einheit von x.
Beispiel: „Ein Sticker kostet 1,50 € pro Stück" → x = Anzahl Sticker, f(x) = Kosten.
Beispiel: „Strom kostet 0,30 € pro kWh" → x = kWh, f(x) = Kosten.

Schritt 2 · Die 3 Fragen

1Was ist gegeben? Startwert (b) und Veränderung pro Schritt (m).
2Was ist gesucht? Achte auf die Signalwörter (siehe Merkkasten unten).
3Womit rechne ich? Funktionswert, Wert, Nullstelle, Schnittpunkt oder Punktprobe.

Beispiel

„Lena hat 200 Follower und bekommt täglich 40 dazu."
→ x = Zeit in Tagen, f(x) = Anzahl Follower → Startwert b = 200, Veränderung m = 40 → f(x) = 40x + 200

📌 Merkkasten – Signalwörter erkennen

Daran erkennst du, was du rechnen musst.
Das steht im Text (Signalwort) Das berechnest du Mathematischer Ansatz Beispiel
„Wie viel … nach 12 Tagen?" – ein Zeitpunkt (x) ist gegeben einen Funktionswert berechnen Zahl für x einsetzen, f(x) ausrechnen f(12) = 40·12 + 200
„Wann erreicht … genau 500?" – ein Funktionswert (y-Wert) ist gegeben, der x-Wert gesucht einen Wert berechnen f(x) = 500 setzen, nach x auflösen 40x + 200 = 500
„Wann ist … leer / aufgebraucht / 0 / fertig?" eine Nullstelle berechnen f(x) = 0 setzen, nach x auflösen −16x + 240 = 0
„Wann ist … halb voll / halb leer?" einen Wert berechnen f(x) = halber Startwert setzen, nach x auflösen −16x + 240 = 120
„Wann sind beide gleich? / Wann überholt …? / Ab wann lohnt sich …?" einen Schnittpunkt berechnen f(x) = g(x) setzen, nach x auflösen 8x + 60 = 12x + 20
„Liegt Punkt P auf der Geraden? / Stimmt der Wert / die Angabe?" eine Punktprobe machen x-Wert von P einsetzen; ist f(x) = y-Wert von P? f(4) = 2·4+5; ist das 13?
„Stelle die Funktionsgleichung auf" – Startwert & Veränderung gegeben eine Funktionsgleichung aufstellen m und b bestimmen: f(x) = m·x + b Start 200, +40/Tag → 40x+200

Aufgabe B3 · Powerbank beim Zocken 🔋

f(x) = · x +
Die Ladung wird weniger – überlege, welches Vorzeichen die Steigung m dann hat.

Mia und Jana sparen für ein Konzert von Stray Kids.

Mia: M(x) = · x +
Jana: J(x) = · x +
Das schon gesparte Geld ist b, das wöchentlich Gesparte ist m.
M(x) = 8x + 60  und  J(x) = 12x + 20
b) Nach wie vielen Wochen haben beide gleich viel Geld?
Was musst du berechnen?
Es geht um zwei Personen. Was haben Mia und Jana zum gesuchten Zeitpunkt gemeinsam?

8x + 60 = 12x + 20   | − 8x

= 140 € (Probe: J(10) = 12·10+20 = 140 € ✓)

Aufgabe B5 · Abo oder Kaufen? 🎮

Du willst ein beliebtes Game-Paket nutzen. Es gibt zwei Möglichkeiten:
Möglichkeit A: 12 € pro Monat im Abo.
Möglichkeit B: einmalig 60 € kaufen + 4 € pro Monat für Updates.
🔍 Zuerst: Was ist x, was ist f(x)?
x ist …   f(x) ist …
a) Stelle für beide Möglichkeiten eine passende Funktionsgleichung auf.
A(x) = · x +
B(x) = · x +
Bei A gibt es keine Startkosten. Bei B ist der einmalige Kauf der Startwert b.
A(x) = 12x  und  B(x) = 4x + 60
b) Bei wie vielen Monaten kosten beide Möglichkeiten gleich viel?
Was musst du berechnen?
Wann ist es egal, welche Möglichkeit du wählst? Was haben beide dann gemeinsam?
Setze die beiden Kostenfunktionen gleich: A(x) = B(x).
12x = 4x + 60   | − 4x
8x = 60   | : 8
x = 7,5 Monate
c) Was lohnt sich, wenn du das Game länger als 8 Monate nutzt?
Rechne für eine Monatszahl nach Monat 8 beide Kosten aus und vergleiche, welche kleiner ist.
A(10) = 120 €  ·  B(10) = 4·10+60 = 100 €
Ab Monat 8 ist B günstiger → Möglichkeit B (kaufen)
